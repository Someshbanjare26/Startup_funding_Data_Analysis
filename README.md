# India Startup Funding Analysis (2000–2025)

An end-to-end Data Analytics project analyzing startup funding trends in India using Python, MySQL, and Power BI.

---

##  Project Overview

This project analyzes startup funding data from 2000 to 2025 to uncover trends in investments, industries, cities, and funding rounds. The goal is to demonstrate a complete analytics pipeline from raw data to interactive dashboards.

---

##  Tools & Technologies

- Python (Pandas, NumPy)
- MySQL
- Power BI
- Jupyter Notebook

---

##  Project Structure


---

##  Workflow

1. Raw datasets collected and merged
2. Data cleaning & preprocessing using Python
3. Clean data stored in MySQL database
4. Data connected to Power BI
5. Data modeling using star schema
6. Interactive dashboards created
7. Dashboard published to Power BI Service

---

##  Dashboard Features

- Funding trends over time
- City-wise and industry-wise funding
- Funding round distribution
- Investor-focused analysis
- Professional startup-themed UI

---

##  Key Insights

- Funding growth accelerates post-2015
- Metro cities dominate investments
- Technology-driven startups attract maximum funding
- Large investments occur in later funding rounds
